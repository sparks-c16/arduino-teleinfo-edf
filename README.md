# arduino-teleinfo-edf



## Liste des composants

- Arduino Nano
- 1 module radio émetteur 433 MHz avec antenne
- 1 connecteur à vis 2 pôles
- 1 optocoupleur (LTV-814) 
- 2 resistances 10kOhm
- 1 resistance 4,7kOhm
- 1 MOSFET-N (BS170)
